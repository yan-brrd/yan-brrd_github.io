// Objective

I'm Adrian Gian Barredo, an aspiring SOC Analyst building hands-on detection and response skills outside the classroom. Over 15 intensive days I set up a home lab to practice the full lifecycle of a security incident — from raw packet capture to a written report — using the same categories of tooling found in real SOC environments.

// Background

I'm a graduating Computer Engineering student looking to broaden my scope beyond what's covered in a typical curriculum. Cybersecurity caught my interest early on, and this lab was my chance to move past theory and actually get hands-on with the tools and workflows a SOC analyst uses day to day. I'm aiming to break into an entry-level SOC Analyst or Security Operations role where I can keep building on this foundation.
